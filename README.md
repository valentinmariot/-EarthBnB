### Initialisation
• 1 :      
````
cd wordpress/wp-content/themes/earthbnbTheme
````
• 2 :         
````
npm i
````

### Lancement

• Docker :      
````
docker-compose up -d
````
• SCSS => CSS :      
````
npm run watch
````
• SCSS => CSS compressé :      
````
npm run prod
````
